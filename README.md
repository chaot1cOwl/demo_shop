# Демо онлайн-магазин

Используемые технологии: 
PHP 
jQuery
HTML + CSS
Bootstrap

Для развертывания копии у себя: 

1. Клонировать репозиторий на локальный компьютер. Для этого выполнить ОДНО ИЗ следующего: 
- Скачать Zip-архив с файлами, распаковать в нужном месте
- Выполнить git clone в нужном расположении

Запоминаем расположение, куда разместили файлы, далее оно будет обозначаться как DIRECTORYPATH

2. Создать БД из файла резервной копии shop_db.sql.

3. Создать файл api/config.php по шаблону (config-template.php), указать данные для подключения к вашей БД.


Опционально: 
1. Добавить виртуальный хост.
Для xampp: открываем на редактирование файл xampp/apache/conf/extra/httpd-vhosts.conf, в конец добавляем

<VirtualHost LOCALHOSTNAME:80>
    ServerAdmin webmaster@LOCALHOSTNAME
    DocumentRoot "DIRECTORYPATH"
    ServerName LOCALHOSTNAME
    ErrorLog "error.log"
    CustomLog "access.log" common
</VirtualHost>

Вместо LOCALHOSTNAME указываем адрес, по которому сайт будет доступен в локальной сети (например, myshop.local). Вместо DIRECTORYPATH указываем полный путь к каталогу, где размещены файлы (например C:/xampp/htdocs/shop). 
Сохраняем изменения.

Для других веб-серверов конфигурация аналогичная, смотреть спецификацию для своего сервера.

2. Добавить переадресацию на виртуальный хост, указав его в файле hosts.
Для Windows: Открыть с правами администратора Блокнот (находим его в меню Пуск, жмем правой кнопкой мыши и выбираем Запуск от имени администратора). Открыть в этом блокноте файл C:\Windows\System32\drivers\etc\hosts. В конец файла добавить строку 127.0.0.1 LOCALHOSTNAME 

Для других ОС аналогичная настройка, смотреть спецификацию для своей ОС.
